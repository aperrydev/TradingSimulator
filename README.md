
# TradingSimulator

A console-based stock trading simulator built in Java with live market data,
portfolio tracking, and bank-style account management.

## Features
- Live stock prices via Alpha Vantage API
- Buy and sell stocks with real ticker symbols
- Portfolio tracking with share counts
- Bank-style account with deposit, withdraw, and balance tracking
- Starting bonus of $10,000 to simulate with

## Setup
1. Get a free API key at [alphavantage.co](https://www.alphavantage.co)
2. In `StockFetcher.java`, replace `YOUR_API_KEY_HERE` with your key
3. Run `Main.java`

## Built With
- Java
- Alpha Vantage API
